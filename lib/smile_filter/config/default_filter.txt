-*- 各コマンドの使用例 -*-
有効にするには行頭の半角スペースを削除してください

`うぽつ'を含むコメントを削除
 @cmt うぽつ

全てのコマンドを無効化
 @cmd all

bigコマンドで書かれたコメントを削除
 @cmd big

bigコマンドで書かれたコメントのサイズをsmallへ変更
 @cmd big => small

IDが31415926のユーザーのコメントを削除
 @uid 31415926

`倍速'単独のコメントを削除
 @reg /\A倍速\z/

全角半角大文字小文字問わず4つ以上連続する`w'を`www'に置換
 @reg /[wｗ]{4,}/i, 'www'

パソコン以外からのコメントをblackかつsmallに変更
 @evl (chat.color = :black; chat.size = :small) unless chat.from_pc?

プレミアム会員以外のコメントを削除
 @evl chat.clear unless chat.premium?

コメント末尾に次の情報を付加
  投稿者コメントなら`F'
  プレミアム会員なら`P'
  見えないコメントなら可視化した上で`I'
  184ではないなら投稿者コメントなら`O'
  削除されていれば可能なら可視化した上で`D'
  ニコられた回数が取得できる状態ならその正の整数
  NGスコアがついていればその負の整数
  その他、見た目から判断できない全てのコマンド
 @dbg show_nontrivial_parameters

使用された全てのコマンドをコメント末尾に付加
 @dbg show_all_commands(chat)

`@cmt'を使用時に特定の文字を無視
次の例では`ほげほげ'や`ふがふが'の他にも
`ほｗｗｗげｗｗほｗｗｗげ'や`ほ  げ　ほ  げ'、
`ふが/ふが'、 `ふwが ふ/が'も削除される。
 @ign 'wWｗW 　/／'
 @cmt ほげほげ
 @cmt ふがふが
 @ign ''
